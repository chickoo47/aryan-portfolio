## ðŸ“¦ Deployment

### Deploy to Vercel

1. Install Vercel CLI:
```bash
npm install -g vercel
```

2. Navigate to the project directory:
```bash
cd /Users/Ary/Downloads/aryan-portfolio
```

3. Deploy:
```bash
vercel
```

Or simply drag and drop the folder to [Vercel Dashboard](https://vercel.com/new)

### Alternative: GitHub Pages

1. Push to GitHub repository
2. Go to Settings > Pages
3. Select source branch and folder
4. Your site will be live at `https://username.github.io/repo-name`

## ðŸŽ¨ Customization

### Update Personal Information

Edit `index.html` to update:
- Name and contact details
- Education information
- Work experience
- Projects
- Skills and technologies
- Social media links

### Change Colors

Edit CSS variables in `css/style.css`:
```css
:root {
    --primary: #6366f1;
    --secondary: #8b5cf6;
    --accent: #ec4899;
    /* ... */
}
```

### Add Your Photo

Place your photo at `images/profile.jpg` (use the provided image) to automatically fill the hero frame. If the file is missing, a gradient fallback icon appears instead.
